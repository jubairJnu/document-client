
# Document Client






## Live URL

https://document-client.vercel.app/login


## Run Locally

Clone the project

```bash
https://github.com/jubairJnu/document-client.git
```

Go to the project directory

```bash
  document-client
```

Install dependencies

```bash
  npm install
 
```



```bash
  npm run dev
```
